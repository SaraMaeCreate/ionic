WHAT YOU'LL NEED


DOWNLOAD AND SET-UP IONIC


Go to Ionic Frameworks website

  • Click “Get Started” and follow along!

Install Node.js

  • To built command line utilities

Setup Ionic in Windows Command (npm)

  • Install What you need
    • Type "npm install -g cordova ionic gulp" in the terminal
    • cd into destination file (I just put mine into my Documents)
    • In the terminal, create either a blank app or a starter pre-made by ionic (see the site)
      • Example: Type "ionic start myApp sidemenu"
    • Create testing server
      • Type: Ionic Serve
    • Choose Address (I chose local hosting)
    • Now you have a (local) site to view your app in!
    • Example: http://localhost:8100/#/app/playlists
    • cd into folder and type Ionic serve in terminal in order to gain IP for testing

Create a Github Repository from existing folder in Git Bash

  • Create a new Repo on Github (I named mine ionic-test-1)
  • cd into destination file (myApp)
  • Type "git init" in the terminal
    • This initializes an empty Git repository in the myApp folder
  • Finish commands
  • git add .
  • git commit -m 'create repository'
  • git remote add origin https://github.uconn.edu/smp13010/ionic-test-1.git
    • This tells the terminal where to push myApp folder contents
  • git push origin master
