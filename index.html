IONIC CRASH COURSE

Welcome!
In this Demo you'll be learning how to simply setup and begin working on your first Ionic App.
This Demo is meant for PC users who might not exactly know where to begin, and are trying to find the simplist way to initialize Ionic without having to download a bunch of programs to assist in the build.
We'll begin setup within the command terminal.
Once you're setup, you'll be able to use web based programs (I prefer Sublime Text 3) to stylize and Angular to create functions within your first app.
So, let's begin!

WHAT YOU'LL NEED

DOWNLOAD AND SET-UP IONIC


Go to Ionic Frameworks website

  • Click “Get Started” and follow along!

Install Node.js

  • To built command line utilities

Setup Ionic in Windows Command (npm)

  • Install What you need
    • Type "npm install -g cordova ionic gulp" in the terminal
    • cd into destination file (I just put mine into my Documents)
    • In the terminal, create either a blank app or a starter pre-made by ionic (see the site)
      • Example: Type "ionic start myApp sidemenu"

Create testing server

  • Type: Ionic Serve
    • Choose Address (I chose local hosting)
    • Now you have a (local) site to view your app in!
    • Example: http://localhost:8100/#/app/playlists
    • cd into folder and type Ionic serve in terminal in order to gain IP for testing

Create a Github Repository from existing folder in Git Bash

  • Create a new Repo on Github (I named mine ionic-test-1)
  • cd into destination file (myApp)
  • Type "git init" in the terminal
    • This initializes an empty Git repository in the myApp folder
  • Finish commands
    • git add .
    • git commit -m 'create repository'
    • git remote add origin https://github.uconn.edu/smp13010/ionic-test-1.git
      • This tells the terminal where to push myApp folder contents
    • git push origin master

Customization


Use Ionics website to learn how to change your code within your web text editor of choice. Seeing as this is your first Ionic App, let's overview the 3 major components you can edit.

  1. Components
    • Alter premade Ionic default mode configuration properties
  2. SASS Variables
    • Ionic uses SASS for simple and organized customization of colors and variables!
  3. Angular
    • Angular is a structural framework that uses HTML to dynamically create web based apps

Launch!


Once you've completed all of the above steps and are satisfied with your app, you'll be ready to launch! All you'll need is hosting to make your app live; you can't actually view your web based app if it's not on a hosting site: hence the Ionic serve section on local hosting to view app edits.

Launching your app on Google Play Store or IOS is a whole nother process, which ionic provides steps on in their Publishing section of their site.
